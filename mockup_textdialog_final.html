# js变量和函数

阅读使用 Rhino v1.8.1 作为JavaScript引擎，用于在JS中调用Java类和方法。

## 内置变量

在书源规则中使用 @js <js> {{}} 可使用JavaScript调用阅读内置的类和方法。

java - 当前类
baseUrl - 当前url
result - 上一步的结果
book - 书籍类
chapter - 章节类
source - 基础书源类
cookie - cookie操作类
cache - 缓存操作类

## 常用函数

网络请求：

java.ajax(urlStr) // GET请求
java.post(url, body, headerMap) // POST请求
java.connect(urlStr, header) // 完整请求
